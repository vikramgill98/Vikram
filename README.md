# Dreams Into Reality

A static luxury automotive lifestyle experience built for GitHub Pages.

## Run Locally

```sh
python3 -m http.server 4173
```

Open `http://localhost:4173`.

## Deploy

Push this folder to a GitHub repository and enable GitHub Pages from the repository root. No build step is required.

## Notes

The experience uses GSAP, ScrollTrigger, Lucide icons, and remote royalty-free media from Pixabay, Pexels, and Unsplash. Sound is generated in the browser after a user gesture to comply with autoplay restrictions.
